# Installer Notes

This folder contains helper launch files for easier local use.

## Launcher

- `launch-vtu-automator.bat`

## What It Does

The launcher:

- finds Edge or Chrome
- opens a separate browser profile for VTU Automator
- loads the extension from `../extension`
- opens VTU on the `student-diary` page

## How To Run

1. Extract the repository or release package
2. Open the `installer/` folder
3. Double-click `launch-vtu-automator.bat`
4. Log in to VTU in the launched browser window

## Important Notes

- this is not a store install
- users still need the extracted repo on disk
- users still need to log in manually
- supported browsers:
  - Microsoft Edge
  - Google Chrome
